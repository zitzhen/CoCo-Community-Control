# 【全局,安全,自定义】自定义COCO
本控件通过CSS通配选择器修改所有元素的样式，并且有安全的“禁用所有修改”选项，如网页白屏，可能是CSS样式造成的，可尝试在F12中删除带有通配(*)选择器的样式。
