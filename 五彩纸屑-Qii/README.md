<h2>使用方法</h2>
</div>

你可以在任何时候调用控件来释放礼花。

![积木：释放礼花](https://cc.zitzhen.cn/control/五彩纸屑-Qii/images/2.png)

有很多参数，保持默认即可。`X轴`和`Y轴`用来调整释放礼花的位置，范围在`0.0 ~ 1.0`。
X轴上`0`表示屏幕左侧，`1`表示屏幕右侧。
Y轴上`0`表示屏幕顶部，`1`表示屏幕底部。
