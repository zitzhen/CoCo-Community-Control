<h2>更新说明</h2>
</div>

**版本：1.2.0**
新增 获取控件ID 积木，调用后返回当前控件的 ID。
支持 任意控件 功能。

**版本：1.1.0**
新增 禁用状态 参数，禁用后用户将无法控制，但可以通过积木来设置。
新增 是否正在滑动 积木，可以在循环中判断此值来禁止程序控制滑条。
